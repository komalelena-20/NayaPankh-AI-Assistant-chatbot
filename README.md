# NayaPankh-AI-Assistant
NayaPankh AI Assistant

NayaPankh is an AI-powered chatbot developed using Python, Flask, HTML, CSS, JavaScript, Bootstrap, and Google Gemini API.

Features:
• Real-time AI chat interface
• Gemini AI integration
• Dark/Light mode
• Responsive UI
• Typing indicator
• Enter key support
• Clear chat functionality

Technology Stack:
Frontend: HTML, CSS, Bootstrap, JavaScript
Backend: Python, Flask
AI Model: Google Gemini API
